# Stock Trading Platform

## CodeAlpha Java Programming Internship - Task 2

### Project Overview

The Stock Trading Platform is a Java-based console application that simulates a basic stock trading environment. Users can view market data, buy and sell stocks, and track their portfolio performance.

The project is developed using Object-Oriented Programming (OOP) principles and demonstrates the use of classes, objects, collections, and user interaction through a console interface.

### Features

* Display available stocks and market prices
* Buy stocks using available account balance
* Sell owned stocks
* Portfolio management and tracking
* Account balance updates after transactions
* OOP-based design using separate classes
* Easy-to-understand console interface

### Technologies Used

* Java
* Object-Oriented Programming (OOP)
* Java Collections Framework
* Scanner Class for User Input
